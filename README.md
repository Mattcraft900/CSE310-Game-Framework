# Asteroids Clone (Matthew Hilton)

A fun starter project to acquaint myself with the python Arcade library. See how long you can last before getting hit by oncoming asteroids!

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download this directory. The important files are asteroids-clone.py and the images folder.
2. Ensure that you have python 3.7 or later installed, and Arcade 3.3.3 or higher.
3. Navigate to this directory in the CLI of your choice and run `python asteroids-clone.py`. The application window should open and the game will begin automatically.

Instructions for using the software:

- Use WASD or the arrow keys to move your ship around the screen. Traveling past the edge of the screen will wrap you back around to the other side (this is also true for bullets and asteroids).
- Hold left SHIFT while moving to "boost", giving yourself a higher acceleration. Be careful not to lose control of your ship!
- Asteroids will spawn into the arena and stay there unless you shoot them. Bigger asteroids will split into smaller ones when destroyed, so watch out!
- Press the space bar to fire bullets at oncoming asteroids. There is a half-second cooldown between every shot you take, so aim well!
- Difficulty will gradually increase the more asteroids you destroy. See how long you can survive!
